# Changelog

All notable changes to this project will be documented in this file.

## [26.4.0] - 2026-04-01

### Added

- Added favorites for automation report cards.
- Added pagination and a list view toggle for automation reports.
- Added back-to-top controls in the news and automation views.
- Added web search support for automation projects and discussions, including the related settings and streaming output flow.
- Added custom context menus for text inputs in release builds.

### Changed

- Renamed the Creative workspace to Automation and updated related labels and iconography.
- Tightened layout spacing across the app.
- Updated automation project article defaults to 10 minimum and 200 maximum.

## [26.3.3] - 2026-03-26

### Added

- Added `min_articles_per_card` support for creative projects.

### Changed

- Flattened UI components and refined interactions.

### Fixed

- Fixed a race condition in unread counts after source refresh.
- Fixed early truncation in the news source list.

## [26.3.2] - 2026-03-25

### Added

- Added sidebar unread indicators and synced news badge updates.

### Changed

- Improved sidebar UI layout and styling.

## [26.3.1] - 2026-03-24

### Added

- Added OPML import support for sources.
- Added OPML export from the source manager menu.
- Added Windows x64 and Windows ARM64 release builds.

### Changed

- Updated the release workflow so hyphenated version tags are treated as prereleases.

## [26.3.0] - 2026-03-23

### Added

- Initial version.
